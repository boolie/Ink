Calendar Samples - InkJS

Calendar #1 - Default Markup

This is just a calendar with the default markup. You can select any date you like, and there is no upper or lower limit to what dates you can select.

JS: You can also see an example usage of the onSetDate callback, which we use here to console.log

Calendar #2 - Sparse valid dates in a sea of invalid dates.

You can only select the January 3rd and January 6th of each year in the calendar above. When you click the "next" button, you are taken to January 3rd, next year. This is done by passing the nextValidDateFn and prevValidDateFn options

Calendar #3 - Valid and invalid dates.

You can only select days which divide by 3, and in months which are not February, and not fridays either.

Note that the prevValidDateFn and nextValidDateFn were not implemented, and so we still see February even though there are no valid dates there. This is intentional in this case.
